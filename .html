Acronyms/Terms

DHCP

Dynamic Host Configuration Protocol. How an internet client gets its configuration information in order to connect to the internet. The DHCP client (e.g. your comp) queries a DHCP server for info (e.g. an IP address) and then uses that to make web requests.

SOAP

Simple Object Access Protocol. A protocol for accessing resources on the Internet. Uses XML

REST

Representational State Transfer. A web architecture, or series of standards that outline how one should organize HTTP requests (sort of CRUD). It's based on the idea that you make requests when you want to transition state. Not as strict as SOAP, doesn't require a special header.

ETL

Extract Transform Load. The process of extracting data from a source, transforming it to fit new organizational needs, and loading it into a new system.

SAN

Storage Area Network. Basically a shared storage space to access shared resources. A general term that encompasses Shared disks, filesystems, etc.

DMZ

comes from "Demilitarized Zone." Also known as perimeter networking, its a physical or logical subnetwork that contains and exposes external-facing services to a larger, potentially untrusted network (e.g. API).

Honeypot

A trap set to catch attempted hacking, usually by pretending an area is part of a network, when actually it is monitored

XSS

Cross site scripting. An exploitation of a security vulnerability that allows an attacker to inject client-side code into an application. (e.g. not escaping html in a form => )
